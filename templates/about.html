About This Demo

Disclaimer

While I do work for MINDBODY, this project was written entirely outside of my day job. The code may not fit all convensions and proven patterns, as it was a learning project for me as much as it was a demonstration for any who might find it useful. If someone out there is a Django pro with a MINDBODY API account, please feel free to improve on it and send pull requests.

Credits and Sources

This demo site uses Django 1.8 as the web framework, and Python 3 as the base language.

The project's source code can be found here, and the work is free for just about any use. Please read the details of the MIT License to learn more.

The original HTML/CSS template used for this site, entitled Progressus, was obtained from gettemplate - kudos to the designer. It has been modified by me to fit the needs of this project, and is licensed under Creative Commons 3.0.

Getting an API Account

Finally, to actually make any use of this project, you're going to need to have access to the MINDBODY SOAP API. Once signed up, using this project is as easy as plugging your source credentials into the right place.
